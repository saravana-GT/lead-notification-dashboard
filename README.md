# 📢 Lead Notification Dashboard

A premium, modern, and fully responsive **Lead Notification Dashboard** designed to display marketing leads and conversions in real-time. Built with pure HTML and custom CSS.

---

## ✨ Features
- **Grid Layout & Glassmorphism:** Clean, modern cards displaying new registrations, payment status, and user sources.
- **Real-Time Notification Cards:** Animated pop-ups showcasing live lead statuses.
- **Responsive Web Design:** Adapts smoothly to mobile, tablet, and desktop screens.
- **Micro-Animations:** Hover states, scale transitions, and active button effects.

## 🛠️ Tech Stack
- **Structure:** HTML5 Semantic Elements
- **Styling:** Vanilla CSS3 (Custom properties/variables, Flexbox, Grid Layouts)

## 🚀 How to Run
1. **Clone this repository:**
   ```bash
   git clone https://github.com/saravana-GT/lead-notification-dashboard.git
   ```
2. **Open in Browser:**
   - Simply double-click the `index.html` file in the root folder, or
   - Use the **VS Code Live Server** extension to launch a local development server for automatic reloading.
